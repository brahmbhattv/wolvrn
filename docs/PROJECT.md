\## Terminal \& File System Cheat Sheet (Windows + Git Bash)

\*\*Navigation\*\*

\- `pwd` — show current folder

\- `ls -la` — list files (incl. hidden)

\- `cd <folder>` / `cd ..` — into/out of a folder



\*\*Files \& Folders\*\*

\- `mkdir <name>` — create folder

\- `echo "text" > file.txt` — create file with text

\- `cat file.txt` (Bash) / `type file.txt` (CMD) — view file

\- `cp a.txt b.txt` (Bash) / `copy a.txt b.txt` (CMD) — copy

\- `mv a.txt b.txt` (Bash) / `move a.txt b.txt` (CMD) — move/rename

\- `rm -f file.txt` (Bash) / `del file.txt` (CMD) — delete file

\- `rm -rf folder` (Bash) / `rmdir /s folder` (CMD) — delete folder + contents



\*\*Search\*\*

\- `grep -R "text" .` — search recursively for text



\*\*Node/Next\*\*

\- `node -v`, `npm -v` — versions

\- `npm run dev` — start Next.js dev server



\*\*Git Essentials\*\*

\- `git status` — what changed

\- `git add <path>` — stage changes

\- `git commit -m "message"` — save snapshot

\- `git log --oneline --graph` — history
